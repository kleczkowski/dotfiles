# dotfiles

Here is my dotfiles repository with essential config for Unix systems.

## Installing

### Oh My Zsh
Go [here](https://ohmyz.sh/) for installation guide.

### direnv 
Follow [these](https://direnv.net/docs/installation.html)
installtion instructions.

### GHCUp
Go [here](https://www.haskell.org/ghcup/) for installation guide.

Install Haskell Stack and Haskell Language Server on machine.

### Node Version Manager
Go [here](https://github.com/nvm-sh/nvm#installing-and-updating)
for installation guide.

### NeoVim
Install using brew or PPA repository (WSL2).
Install [vim-plug](https://github.com/junegunn/vim-plug#installation).

For Ubuntu:
```
$ sudo apt-add-repository ppa:neovim-ppa/stable
$ sudo apt update
$ sudo apt install neovim
```

For MacOS:

```
$ brew install neovim
```
